Supported locales

The following table lists the locales supported by Derby.

Use the territory=ll_CC URL attribute to set the Derby locale.

Table 1. Supported locales
Locale territory=ll_CC Setting
Chinese (Simplified) zh_CN
Chinese (Traditional) zh_TW
Czech cs
French fr
German de_DE
Hungarian hu
Italian it
Japanese ja_JP
Korean ko_KR
Polish pl
Portuguese (Brazilian) pt_BR
Russian ru
Spanish es
